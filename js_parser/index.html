ℹ️ Информация о работе с Backend сервисом

Frontend работает на GitHub Pages, парсинг выполняется на backend сервере.

Архитектура:

  • Frontend: GitHub Pages (интерфейс)
  • Backend: localhost:8000 (парсинг и обработка)
  • Файлы сохраняются на backend сервере
  • Скачивание через API backend сервиса

Требования:

  • Backend сервис должен быть запущен на localhost:8000
  • Python 3.8+ с установленными зависимостями
  • Chrome/Chromium браузер для парсинга
  • Стабильное интернет-соединение
